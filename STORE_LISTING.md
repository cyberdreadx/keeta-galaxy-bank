## Detailed Description

**Yoda Wallet - The Intergalactic Wallet for Web3**

Yoda Wallet is a powerful, secure, and user-friendly Web3 wallet designed for the Keeta L1 and Base networks. Whether you're managing cryptocurrencies, collecting NFTs, or interacting with decentralized applications (dApps), Yoda Wallet provides a seamless experience.

### üöÄ Key Features

**Multi-Chain Support**
- Full support for Keeta L1 Network (Mainnet & Testnet)
- Native integration with Base Network (Ethereum Layer 2)
- Seamless cross-chain asset management

**Asset Management**
- View real-time balances for KTA, ETH, and all supported tokens
- Track NFT collections with beautiful gallery view
- Live price feeds and USD value conversions
- Hidden token management for clutter-free portfolio

**Advanced Account System**
- Multiple account types: Checking, Savings, and Custom accounts
- Internal transfers between your own accounts
- HD wallet with hierarchical deterministic key generation
- Import/export wallet functionality

**dApp Connectivity**
- Compatible with any Keeta or Base dApp
- Standard Web3 provider injection (`window.yoda`, `window.ethereum`)
- Secure connection approval flow
- Manage connected sites easily

**NFT Support**
- View and manage your NFT collection
- IPFS image support with automatic gateway conversion
- Direct links to explorers and marketplaces
- Beautiful grid and list view modes

**Bridge & Swap**
- Built-in bridge for Base ‚Üî Keeta transfers
- Token swap functionality
- Buy crypto with fiat (on-ramp support)

**Security First**
- Non-custodial: You control your keys
- Encrypted local storage
- Manual transaction confirmation for every action
- Open-source and auditable code

**Beautiful UI**
- Star Wars-inspired cyberpunk design
- Smooth animations and transitions
- Responsive layout optimized for 400x600 popup
- Dark mode by default
